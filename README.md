## ✨ 프로젝트 기능
1. 젤리 정보 
: 젤리에 대한 영양정보와 맛과 식감에 대한 점수, 다른 사람들이 남긴 젤리 평점을 확인해 젤리 선택에 도움을 줍니다.

2. 젤비티아이
: 나와 성향이 가장 비슷한 곰 젤리는 무엇일까 확인하고, 나와 비슷한 유형의 사람들은 어떤 젤리를 선호하는지 알 수 있습니다.

3. 회원 게시판
: 젤리에 대한 다양한 이야기를 나눌 수 있습니다.
